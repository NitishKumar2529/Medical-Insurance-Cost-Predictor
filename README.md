# Medical-Insurance-Cost-Predictor

This project is a Medical Insurance Cost Prediction System built using Python, Flask, HTML, and CSS, designed to estimate an individual’s medical insurance charges based on key personal and health-related inputs such as age, BMI, smoking status, number of dependents, and region. The backend is developed with Flask, which handles user requests, processes input data, and integrates a trained machine learning model to generate accurate predictions. The frontend is created using HTML and CSS, providing a clean and user-friendly interface for input collection and result display. The application runs on a local server (localhost), making it lightweight and easy to test or deploy. This project demonstrates the practical implementation of machine learning in real-world healthcare cost estimation, along with full-stack development skills
